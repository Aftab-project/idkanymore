## Image Guidelines

### **Quality Standards:**
- Use professional, high-resolution images
- Ensure proper lighting and composition
- Images should reflect the luxury and professionalism of your brand
- Maintain consistent style across all images

### **File Format:**
- **JPG/JPEG** for photographs (hero banner, treatment images)
- **PNG** for logos and graphics with transparency
- **WebP** (optional) for better performance

### **Optimization:**
- Compress images for web use (recommended tool: TinyPNG, ImageOptim)
- Target file sizes: Hero banner <500KB, Treatment images <200KB each
- Maintain aspect ratio to avoid distortion

### **Color Palette:**
Ensure images complement the website's earthy color scheme:
- Brown (#8B6F47)
- Beige/Gold (#C9A87C)
- Gold Accents (#D4AF37)
- Light beige backgrounds (#F5F1E8)

## Placeholder Behavior

Currently, the website uses CSS gradients as placeholders for missing images. When you add actual images:

1. Save images with the exact filenames listed above
2. Place them in this `/images/` directory
3. Refresh the website to see your images load automatically
4. No code changes required!
